## 2022년 멋쟁이사자처럼 한국외대(서울) 신입회원 지원서 작성 웹사이트
---
2022년 2월 23일부터 3월 4일까지 진행된 멋쟁이사자처럼 한국외대(서울)의 신입회원 모집을 위한 웹사이트 제작 프로젝트입니다.
